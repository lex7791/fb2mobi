Конвертор из FB2 в Mobipocket.

Версия 1.1.0 /24.04.2010/
Автор: Sk0rp
Страница программы: http://code.google.com/p/fb2mobi/

Для работы должен быть установлен .NET Framework 2.0, а также потребуется бесплатный компилятор в Mobipocket, взять его старую версию можно отсюда: http://www.mobipocket.com/dev/default.asp
Для версии начиная с 1.1.0 нужен свежий компилятор KindleGen for Windows (2003, XP, Vista, 7), брать отсюда: http://www.amazon.com/gp/feature.html?docId=1000234621

Использование:
Распаковать fb2mobi.zip, в эту же папку положить загруженный и распакованный kindlegen.exe (для версии 1.0.1 mobigen.exe).
Для конвертирования передавать fb2 файл в fb2mobi.exe как параметр.
Например можно кликнуть на fb2 правой кнопкой, выбрать "Открыть как..." и для открытия указать fb2mobi.exe. Можно fb2 файл перетащить мышкой и кинуть на fb2mobi.exe

Часто задаваемый вопрос:
-Можно ли добавить обработку сразу папки с файлами
-Можно, но лень, это и так легко можно сделать стандартными средствами из командной строки, например:
for %a in (fb2folder\*.fb2) do fb2mobi.exe "%~fa"
обработает все fb2 файлы в папке fb2folder
