Конвертер командной строки из fb2 в Mobipocket и в Kindle.

Версия 2.0.0b /13.02.2011/
Автор: Sk0rp
Страница проекта: http://code.google.com/p/fb2mobi/

Для работы должен быть установлен .NET Framework 2.0
А также нужен свежий компилятор KindleGen for Windows (2003, XP, Vista, 7), брать отсюда: http://www.amazon.com/gp/feature.html?docId=1000234621

Использование:
Распаковать fb2mobi.zip, в эту же папку положить загруженный и распакованный kindlegen.exe
Для конвертирования передавать fb2 файл в fb2mobi.exe как параметр.
Например можно кликнуть на fb2 правой кнопкой, выбрать "Открыть как..." и для открытия указать fb2mobi.exe. Можно fb2 файл перетащить мышкой и кинуть на fb2mobi.exe

Ключи командной строки

Usage: fb2mobi [] [{-,/,--}param]
-nc Не сжимать конечный файл. Это увеличивает скорость работы и размер.
-cl Удалить паку с промежуточными файлами.
-us Использовать имя исходного файла как имя конечного.
-nt Не транслитирировать имя конечного файла.

Часто задаваемый вопрос:

-Можно ли добавить обработку сразу папки с файлами
-Можно, но лень, это и так легко можно сделать стандартными средствами из командной строки, например:
for %a in (fb2folder\*.fb2) do fb2mobi.exe "%~fa"
обработает все fb2 файлы в папке fb2folder
